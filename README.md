# Thực hành Core Components – React Native (Expo)

## Mô tả

Bài tập thực hành sử dụng các Core Components trong React Native để xây dựng màn hình đăng nhập bằng số điện thoại.
Ứng dụng được xây dựng bằng Expo, giao diện đơn giản, đúng theo mẫu đã cho.

## Yêu cầu bài tập

* Sử dụng React Native (Expo)
* Sử dụng các Core Components
* Giao diện giống hình minh họa
* Nộp bài dưới dạng link GitHub (có README.md)

## Các Core Components đã sử dụng

* View
* Text
* TextInput
* TouchableOpacity
* KeyboardAvoidingView
* StyleSheet

## Chức năng

* Hiển thị màn hình đăng nhập
* Nhập số điện thoại bằng bàn phím số
* Nút “Tiếp tục” (chưa xử lý logic)

## Hướng dẫn chạy project

### Bước 1: Cài đặt thư viện

```bash
cd my-app
```

### Bước 2: Chạy ứng dụng

```bash
npm start
```

Sau đó:

* Quét QR bằng ứng dụng Expo Go trên điện thoại
  hoặc
* Chạy trên Android Emulator (nếu đã cài Android Studio)
